    时信魔方

    时信魔方是面向开发者的实时协作开发框架。帮助开发者快速、高效的在项目中集成实时协作能力。

    • 即时消息（Instant Messaging / IM）。
    • 实时多人语音/多人视频（Multipoint RTC）。
    • 超大规模(100+)视频会议 （Video Conference）。支持会议控制、演讲模式，自定义 MCU 和 SFU 布局等。
    • 群组管理（Group management）。
    • 共享桌面（Remote Desktop Sharing）。
    • 云端文件存储（Cloud File Storage）。
    • 实时白板（Realtime Whiteboard）。
    • 视频直播（Live video）。
    • 在线文档协作（Online Document Collaboration）。支持 Word、PowerPoint、Excel 等主流格式文多人在写协作。
    • 支持安全与运维管理（Operation and Maintenance management）。
